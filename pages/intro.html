Quiz : Entrepôt de Données Moderne

1. Quelle est la principale différence entre un data lake et un data warehouse ?

  • a) Le data warehouse stocke uniquement des images
  • b) Le data lake nettoie automatiquement les données
  • c) Le data warehouse impose un schéma structuré, contrairement au data lake
  • d) Le data lake est plus rapide que le data warehouse
Réponse incorrecte. La bonne réponse est : c) Le data warehouse impose un schéma structuré.

2. Pourquoi un data warehouse moderne doit-il être « serverless » ?

  • a) Pour mieux fonctionner hors ligne
  • b) Pour éviter la gestion des clusters et index
  • c) Pour limiter les utilisateurs
  • d) Pour réduire la bande passante
Réponse incorrecte. La bonne réponse est : b) Pour éviter la gestion des clusters et index.

3. Pourquoi la visualisation est-elle essentielle dans un data warehouse ?

  • a) Pour produire automatiquement des vidéos
  • b) Pour décorer l’interface utilisateur
  • c) Pour permettre l’analyse et la prise de décision rapide
  • d) Pour stocker des images satellites
Réponse incorrecte. La bonne réponse est : c) Pour permettre l’analyse et la prise de décision rapide.

4. Quel est l’objectif principal d’un data warehouse par rapport à un data lake ?

  • a) Archiver des fichiers anciens
  • b) Stocker les données brutes uniquement
  • c) Fournir des données propres, cohérentes et interrogeables
  • d) Exécuter des scripts Python sur des fichiers CSV
Réponse incorrecte. La bonne réponse est : c) Fournir des données propres, cohérentes et interrogeables.

5. Quelles fonctionnalités doit offrir un data warehouse moderne ?

  • a) Calcul distribué uniquement
  • b) Requêtes lentes et archivage
  • c) Visualisation, streaming, ML intégré, sécurité d’entreprise
  • d) Traitement en batch uniquement
Réponse incorrecte. La bonne réponse est : c) Visualisation, streaming, ML intégré, sécurité d’entreprise.
